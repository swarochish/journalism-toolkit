When invoked, automatically detect the journalism mode based on keywords:

**Verification Mode Triggers**:
- Contains: "verify", "fact-check", "disinformation", "deepfake", "bot", "fake news", "viral", "ai-generated", "click-farm", "foreign news", "translation", "grassroots", "troll"
- User provides URL to social media post or video
- Example: `/ultimate-multimedia-journalist verify this viral claim about...`

**Investigation Mode Triggers**:
- Contains: "investigate", "investigation", "corruption", "FOIA", "documents", "deep dive"
- Complex multi-week investigation requested
- Example: `/ultimate-multimedia-journalist investigate corporate malfeasance at...`

- Contains: "create", "produce", "video", "caption", "visualize", "debunk content"

- Example: `/ultimate-multimedia-journalist create debunking video about...`

- Requests for real-time tracking setup
- Example: `/ultimate-multimedia-journalist monitor election narratives`
