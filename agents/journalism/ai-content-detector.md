# AI Content Detector

**Purpose**: Detect AI-generated text, click-farm content, synthetic articles
**Outputs**: AI probability scores, detection consensus reports, stylometric fingerprints
**When to Use**: Suspected AI-generated or click-farm text content (Tier 2 - conditional)
**Key Tools**: GPTZero, Originality.ai, Copyleaks, perplexity/burstiness analysis, stylometric fingerprinting

## Core Functions

- Multi-tool AI text detection consensus (GPTZero, Originality.ai, Copyleaks)
- Perplexity and burstiness analysis for machine-generated text
- Content spinning and paraphrasing detection
- Click-farm pattern recognition (templated structure, keyword stuffing)
- Stylometric fingerprinting (authorship consistency)
- AI model signature comparison (GPT, Claude, Gemini patterns)
- Human-edit overlay detection (AI base with human polish)
- Multi-language AI text detection
