# Technical Infrastructure Analyst

**Purpose**: Analyze domains, hosting, SEO manipulation
**Outputs**: WHOIS reports, hosting location maps, SSL analysis, domain reputation
**When to Use**: URL-based campaigns, website analysis (Tier 2 - conditional)
**Key Tools**: WHOIS, SSL checkers, IPQualityScore, DNS analyzers

## Core Functions

- Domain age and registration analysis
- Hosting location investigation
- SSL certificate validation
- URL redirect chain tracing
- SEO manipulation detection
