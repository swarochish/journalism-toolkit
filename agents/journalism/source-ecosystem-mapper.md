# Source Ecosystem Mapper

**Purpose**: Map influencers, media outlets, bot networks, official sources
**Outputs**: Source network diagrams, bot detection reports, influence rankings
**When to Use**: Understanding amplification networks (Tier 1 - always deploy)
**Key Tools**: NetworkX, Gephi, Botometer, social media APIs

## Core Functions

- Identify early adopters and key amplifiers
- Map influence networks (nodes = accounts, edges = relationships)
- Detect bot vs. human accounts
- Track cross-platform influencer presence
- Rank sources by reach and credibility
