# Grassroots Sincerity Analyst

**Purpose**: Evaluate real human accounts for sincerity, troll-army patterns, and information value
**Outputs**: Sincerity scores (0-100), troll-army indicators, information value assessments
**When to Use**: Real human accounts flagged by bot-network-detector need sincerity evaluation (Tier 2 - conditional)
**Key Tools**: Behavioral analysis, account history profiling, engagement pattern analysis, sentiment consistency scoring

## Core Functions

- Sincerity scoring (0-100) for verified human accounts
- Troll-army pattern detection in real humans (coordinated talking points, shift-based posting)
- Information value assessment (signal vs noise ratio)
- Paid shill indicators (undisclosed sponsorship, sudden topic pivots)
- Ideological consistency analysis (genuine belief vs performative amplification)
- Account behavioral profiling (pre/post radicalization shifts)
- Grassroots willowing (separate high-value grassroots voices from coordinated amplifiers)
- Coordination with bot-network-detector and source-ecosystem-mapper
