# Narrative Timeline Analyst

**Purpose**: Track viral spread, story emergence, peak periods
**Outputs**: Interactive timelines, viral spread maps, peak engagement metrics, Google Trends correlation
**When to Use**: Every verification investigation (Tier 1 - always deploy)
**Key Tools**: Twitter API, Reddit API, TikTok API, YouTube API, Google Trends (pytrends), Plotly, Folium

## Core Functions

- Detect first mention of narrative across platforms
- Track peak trending periods with statistical analysis
- Map geographic spread with heat maps
- Analyze platform-specific virality metrics
- Correlate with search volume trends
- Track dormancy and re-emergence cycles
