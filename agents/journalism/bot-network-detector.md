# Bot Network Detector

**Purpose**: Identify coordinated inauthentic behavior, bot networks
**Outputs**: Bot probability scores, coordination clusters, network graphs
**When to Use**: Suspicious account activity detected (Tier 2 - conditional)
**Key Tools**: Botometer, network analysis, posting pattern detection, scikit-learn

## Core Functions

- Account authenticity scoring
- Posting pattern analysis (entropy, timing)
- Coordinated behavior detection
- Sock puppet identification
- Cross-platform account linking
