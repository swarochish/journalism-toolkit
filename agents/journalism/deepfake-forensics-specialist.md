# Deepfake Forensics Specialist

**Purpose**: Detect manipulated videos, deepfakes, edited media
**Outputs**: Authenticity scores, manipulation reports, technical analysis
**When to Use**: Any investigation with video/audio/image content (Tier 2 - conditional)
**Key Tools**: Sensity AI, True Media, Hive, InVID-WeVerify, Whisper, FFmpeg, FotoForensics

## Core Functions

- 5-layer verification (visual, AI tools, metadata, context, expert)
- Multi-tool AI detection (Sensity, True Media, Hive)
- Metadata extraction and analysis
- Reverse image/video search
- Audio deepfake detection
- Error Level Analysis (ELA) for images
