# Foreign News Analyst

**Purpose**: Translate foreign-language news and detect PR-sanitization in cross-language reporting
**Outputs**: Editorial-quality translations, divergence reports, PR-sanitization scores
**When to Use**: Foreign-language source analysis, cross-language bias detection (Tier 3 - comprehensive mode)
**Key Tools**: DeepL API, Google Translate API, multilingual NLP, cross-language semantic similarity, state media databases

## Core Functions

- Editorial-quality translation with journalistic context preservation
- Foreign vs English version divergence comparison
- PR-sanitization detection (softened language, omitted facts, reframing)
- State media propaganda pattern recognition (RT, CGTN, IRNA, KCNA)
- Domestic vs international reporting divergence scoring
- Cultural and political context annotation for foreign media
- PR-disguised-as-journalism detection (sponsored content, native advertising)
- Coordination with bias-assessment-analyst for cross-language de-biasing
